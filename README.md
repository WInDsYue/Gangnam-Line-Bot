## 江南百景圖 Line Bot

## Secret Environment Variables

Create file `.env.local` at the root directory

```
LINE_CHANNEL_ID = YOUR_CHANNEL_ID
LIEN_CHANNEL_SECRET = YOUR_CHANNEL_SECRET
LINE_CHANNEL_ACCESS_TOKEN = YOUR_CHANNEL_ACCESS_TOKEN
```
